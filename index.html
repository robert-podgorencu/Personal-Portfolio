My recent projects:

Banking System Back-End

This project has been created using Java/SpringBoot and MySQL and it has the following functionalities:

  • Have the possibility to login / logout
  • Be able to have multiple accounts in different currencies
  • Be able to see the available balance for all the accounts
  • Be able to exchange money from a currency to another (hardcoded exchange rate)
  • Be able to make a transaction to another user
  • Be able to request money from another user
  • Be able to see the transactions history (income/ outcome)

Journal Android App

This project is an android application written in Java which has the following functionalities:

  • Have the possibility to login / logout / sign up

  • The data is saved using Google Firebase Firestore

Fullstack Website for Prosumers and Energy Companies

This project is a fullstack web application written in Java/SpringBoot and TypeScript/Angular which has the following functionalities:

  • Two types of users: Company and Prosumers
  • Have the possibility to login / logout / sign up
  • Add posts
  • View the energy generated by the Prosumer's solar panels
  • View the list of posts created by the other users
  • The back-end receives information from an IoT system through HTTP requests
